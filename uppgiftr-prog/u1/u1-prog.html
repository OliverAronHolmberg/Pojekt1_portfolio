Uppgift 1 | Hello World!

Denna uppgift handlar om att komma igång med Python genom att skriva ut text på skärmen, använda citattecken, skapa flera rader, slå ihop strängar och ta emot information från användaren. Målet är att förstå hur print(), strängar och input fungerar i praktiken.

Att arbeta med strängar och citattecken

Textsträngar kan innehålla olika typer av citattecken. Genom att använda rätt syntax går det att inkludera citattecken inom strängar och skapa flerradiga texter med trippelcitattecken.

Grundläggande Utskrifter i python

Print()-funktionen används för att skriva ut text och värden på skärmen. Det är ett grundläggande sätt att visa information och följa programmets flöde.

Ta emot och använda data från användaren

Input()-funktionen möjliggör att programmet kan ta emot inmatning från användaren. Den insamlade informationen kan sedan användas för att skapa dynamiska och interaktiva program.
